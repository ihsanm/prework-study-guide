HTML

  • The Head element contains information about the page
  • The body element contains visible content shown to the user

CSS

  • margin indicates how much space we want outside the element
  • Padding indicates how much space we want around the content inside of an element

Git

  • git status shows what branch we are currently in
  • git checkout -b branch-name creates a new branch and switches to it

Javascipt

  • A function is a set of rules that tells the computer how to perform a certian task
  • A variable is a named container that allows us to store data in our code.
